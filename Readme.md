# Discount Calculator 🧮

A lightweight, user-friendly desktop application built with Python and Tkinter. This tool allows users to quickly calculate the final price of items after applying a percentage-based discount and factoring in the quantity.

## ✨ Features

- Intuitive Interface: Simple, clean layout with a professional blue theme.
- Smart Calculation: Handles original price, discount percentage, and quantity simultaneously.
- Error Handling: Built-in validation to prevent crashes if non-numeric characters are entered.
- Currency Formatting: Automatically formats the result in Rupiah (Rp) with digit grouping.

---

## 🚀 Getting Started

### Prerequisites

Make sure you have **Python 3.x** installed on your system.

- [Download Python](https://www.python.org/downloads/)

### Installation & Execution

1. Clone or Download this repository.
2. Save the code as `main.py`.
3. Open your terminal or command prompt and run:
   ```bash
   python main.py
   ```
